About

2048 is a single-player sliding block puzzle game. You need to move blocks in order to combine the same tiles. The goal is to reach 2048 or a maximum you can.

It is yet another implementation of 2048 game initially written by Gabriele Cirulli.

You can find more information about this game also on Wikipedia.

This version of a game is implemented as open source software. Main repository is located on Github.

Used 3rd party assets:

  • CSS framework Bulma.
  • Icon font Open iconic.
